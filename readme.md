## CoreFit – E-commerce Landing Page

CoreFit is a modern and fully responsive fitness e-commerce landing page converted from Figma into clean, production-ready code. It is designed for gym brands, fitness trainers, and wellness businesses that want to showcase services, trainers, and fitness products in a strong conversion-focused layout.

Built with HTML, CSS, JavaScript, jQuery, Swiper Carousel, and WOW.js animation, the website delivers smooth interactions, engaging animations, and a premium user experience across all devices. It features product showcase sections, trainer profiles, category highlights, strong call-to-action areas, and a professional fitness brand presentation designed to increase user engagement and trust.
